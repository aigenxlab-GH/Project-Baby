================================================================================
WHAT HAPPENS WHEN YOU RUN THE SCRIPT
================================================================================

1. Script scans all 380+ MDX files in content/ directory
2. Categorizes each by type (product, health, parenting, general)
3. Checks if disclaimers already exist (avoids duplicates)
4. Inserts appropriate disclaimers:
   - After first H2 heading (for medical/product)
   - Before Related Articles section (for sources)
5. Reports progress every 50 files
6. Generates CSV audit log: disclaimer_processing.log
7. Prints final statistics

Expected output:
  ✓ Total files processed: 380+
  ✓ Files modified: ~325-330
  ✓ Files skipped: ~50-55 (already have disclaimers)
  ✓ Log file generated: disclaimer_processing.log
