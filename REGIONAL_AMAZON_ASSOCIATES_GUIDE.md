## Step 8: Common Issues & Solutions

### **Problem 1: Accounts Rejected**
- **Reason**: Usually "website doesn't have enough traffic"
- **Solution**: 
  - Reapply after 30 days
  - Wait until you have 1,000+ monthly visitors
  - Update application with current traffic stats

### **Problem 2: Approval Takes 2-4 Weeks**
- **Normal**: Don't worry, just wait
- **Faster approval**: Follow Amazon's guidelines strictly

### **Problem 3: Tracking ID Issues**
- **Not tracking**: Make sure you're using CORRECT tag in URL
- **Wrong commission**: Verify you're using correct tracking ID for each region

### **Problem 4: Links Not Converting**
- **Check**: Users click but don't buy
- **Solution**: This is normal early on, give it 2-3 months
